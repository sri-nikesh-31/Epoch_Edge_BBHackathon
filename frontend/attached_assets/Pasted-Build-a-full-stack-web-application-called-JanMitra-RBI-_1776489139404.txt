Build a full-stack web application called **“JanMitra – RBI Policy Assistant”** focused on democratizing financial access.

## 🎯 Core Purpose

The platform helps users understand RBI policies in simple language, regional languages, and personalized context (job, income, category).

---

## 🏠 1. Home Page

Design a clean modern landing page with:

* Hero section:

  * Title: "Understand RBI Policies in Your Language"
  * Subtitle: "AI-powered assistant for financial awareness"
  * CTA buttons: "Try Chatbot" and "Explore Policies"

* Features section:

  * Explain RBI circulars in simple terms
  * Translate into multiple Indian languages
  * Voice-enabled chatbot
  * Personalized impact (farmers, MSMEs, students, etc.)

* How it works:

  1. Upload / select RBI circular
  2. AI summarizes it
  3. User selects language
  4. System explains impact

* Add clean UI using TailwindCSS

---

## 📊 2. Dashboard (Tables + Graphs)

Create a dashboard page showing:

* RBI policy dataset (table view)

  * Columns: Title, Date, Category, Affected Group

* Graphs:

  * Policies by category (bar chart)
  * Timeline of policies (line chart)
  * Sector impact distribution (pie chart)

Use chart library (Recharts or Chart.js)
