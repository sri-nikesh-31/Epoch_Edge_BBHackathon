# Epoch Edge BBHackathon

## Jan-Mitra RBI Chatbot

This repository contains the completely structured application consisting of both the user-facing interface and the intelligence pipeline.

### Directory Structure

- `frontend/`: Contains all the frontend assets, React/Vite components, and dependencies for the Chatbot application.
- `rag_pipeline/`: Contains the Python-based Retrieval-Augmented Generation (RAG) backend.

### Getting Started

#### Frontend
1. Navigate to the `frontend/` directory.
2. Run `npm install` to install frontend dependencies.
3. Start the dev server using `npm run dev`.

#### RAG Pipeline
1. Navigate to the `rag_pipeline/` directory.
2. Install the necessary dependencies via `pip install -r requirements.txt`.
